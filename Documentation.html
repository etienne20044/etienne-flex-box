Jag stötte på ett stort antal problem när jag skapade min hemsida, det svåraste var att få allt på plats eftersom saker flyttades runt vid kontakt av andra saker. För att lösa det problemet använde jag mig av containers för att lätt kunna se vad som låg vart vilket gjorde att det var lättare för mig att avgöra vart jag inte kunde placera objekt. Jag fick även problem med bilder som försvann för någon anledning, jag vet än inte vad som var problemet men det löste sig när jag bytte ut bilderna. I början av projektet hade jag även svårt att komma på vad jag skulle sälja, jag hade svårt att börja med uppgiften och de bristande kunskaperna om flexbox gjorde saker ännu svårare eftersom jag missade många av lektionerna då vi arbetade med flexbox. Jag lärde mig mycket av denna uppgift.
